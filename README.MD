# REACT-START-PACK
This is boilerplate for react.js with redux.


# WHAT IN THE PACK

-	react v^16
-	sass-loader
-	redux
-	react-router v4
-	webpack v4
-	hot module replacement (HMR)

## HOW TO START
**NPM**

	npm install && npm run dev

**YARN**

	yarn install && yarn dev


## HOW TO START AS PRODUCTION
	
**NPM**

	npm install && npm start

**YARN**

	yarn install && yarn start
